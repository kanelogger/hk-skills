# 财务分析 Skill

## 概述

本技能通过 AKShare（免费数据源）获取 A 股上市公司的利润表财务数据，生成交互式 HTML 可视化报告，分析：
- 毛利率走势
- 各项费用率走势（销售费用率、管理费用率、财务费用率、研发费用率）
- 净利率走势
- 多公司财务指标对比

## 前置要求

```bash
pip install akshare pandas
```

数据来源：AKShare 新浪财经利润表接口（免费）

## 工作流程

### 步骤 1: 获取数据

```bash
python scripts/fetch_financial.py --codes "601012,688599,688223" --output financial_data.json
```

**参数说明**:
- `--codes`: 股票代码，多个用逗号分隔
- `--years`: 回溯年份（默认5年）
- `--output`: 输出 JSON 文件路径

### 步骤 2: 生成分析报告

```bash
python scripts/analyze_financial.py --data financial_data.json --output financial_analysis.html
```

**参数说明**:
- `--data`: 步骤1生成的JSON数据文件
- `--output`: 输出HTML报告路径

### 步骤 3: 查看报告

报告包含：
1. **汇总卡片**: 最新季度、营业收入、毛利率、净利率、变化幅度
2. **图表1**: 三大费用率走势（折线图）
3. **图表2**: 毛利率 vs 净利率（柱状图）
4. **数据表格**: 所有财务指标完整展示

## 指标说明

| 指标 | 计算方式 | 解读 |
|------|---------|------|
| 毛利率 | (营收-成本)/营收 | 产品定价能力 |
| 销售费用率 | 销售费用/营收 | 市场投入效率 |
| 管理费用率 | 管理费用/营收 | 内部管理效率 |
| 财务费用率 | 财务费用/营收 | 融资成本压力 |
| 研发费用率 | 研发费用/营收 | 创新能力投入 |
| 净利率 | 净利润/营收 | 最终盈利水平 |

## 故障排除

- **数据为空**: 检查股票代码是否正确
- **网络错误**: 确保网络连接正常
- **安装失败**: 运行 `pip install akshare pandas`
