# 市值与净利润关系分析

## 概述

本技能生成交互式 HTML 可视化报告，帮助分析：
- 市值与净利润的历史走势对比
- 市盈率(PE)的变化趋势
- 季度环比变化分析
- 市值涨幅与净利润增长的对比
- 市值与盈利的相关性分析

## 前置要求

1. **Python 依赖**:
   ```bash
   pip install pandas numpy
   ```

2. **数据来源说明**:
   - 使用 hexin-ifind-ds-stock-mcp MCP 工具获取真实市场数据
   - 数据源：同花顺 iFinD 金融数据库
   - 覆盖：A 股全部上市公司
   - 市值 = 收盘价 × 流通股本（使用流通股本，更准确反映实际可交易市值）
   - 净利润数据：季报数据，存在披露滞后

## 工作流程

### 步骤 1: 获取数据

使用 `scripts/fetch_data.py` 脚本获取股票数据：

```bash
python scripts/fetch_data.py --codes "601012,688008,300393" --start "2024-01-01" --end "2024-12-31" --output stock_data.json
```

**参数说明**:
- `--codes`: 股票代码，多个用逗号分隔（必填）
- `--start`: 开始日期 YYYY-MM-DD 格式（可选，默认近 3 年）
- `--end`: 结束日期 YYYY-MM-DD 格式（可选，默认今天）
- `--output`: 输出 JSON 文件路径（默认 stock_data.json）

### 步骤 2: 生成分析报告

使用 `scripts/analyze.py` 脚本生成 HTML 可视化报告：

```bash
python scripts/analyze.py --data stock_data.json --output market_cap_analysis.html
```

**参数说明**:
- `--data`: 步骤 1 生成的 JSON 数据文件（默认 stock_data.json）
- `--output`: 输出 HTML 报告路径（默认 market_cap_analysis.html）

### 步骤 3: 查看报告

生成的 HTML 报告包含：
1. **汇总卡片**: 最新市值、最新季度净利润、PE、数据区间
2. **图表 - 市值与净利润走势对比**: 双 Y 轴折线图，市值（左轴）与净利润（右轴）
3. **数据表格**: 完整的季度数据
4. **分析洞察**: 自动生成的文字分析

## 分析维度说明

### 市盈率 (PE)
- PE = 市值 / 净利润（单位统一为亿元）
- PE < 20: 可能被低估
- PE 20-50: 合理区间
- PE > 50: 可能存在估值泡沫风险

### 市值与净利润相关性
- 相关系数 > 0.5: 强正相关
- 相关系数 0.2-0.5: 弱正相关
- 相关系数 -0.2 到 0.2: 基本无关
- 相关系数 < -0.5: 强负相关

## 注意事项

1. **数据来源**: 当前版本使用模拟数据，实际使用时应集成真实数据接口
2. 复权方式: 价格数据为前复权
3. 季报披露: 净利润数据滞后一个季度
4. 负净利润: 对于亏损公司，PE 指标不适用
5. 市值计算: 使用流通股本计算

## 故障排除

- **数据为空**: 检查股票代码格式是否正确（6 位数字）
- **日期格式**: 确保日期格式为 YYYY-MM-DD
- **Python 版本**: 确保使用 Python 3.7+
- **安装失败**: 运行 `pip install pandas numpy`
