# 老梨×光伏产业监控 v1.0

> 光伏产业实时监控工具 | 每天自动推送价格趋势+舆情预警

---

## 一句话介绍

这是一个自动化光伏产业监控工具，每天自动收集多晶硅价格数据、监控光伏组件Top10企业负面舆情，生成分析报告并推送微信。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 价格监控 | 多晶硅价格趋势分析（数据来源：硅业分会） |
| 舆情监控 | 光伏组件Top10企业产能受损风险预警 |
| 自动运行 | 每天08:00自动执行，无需人工干预 |
| 微信推送 | 自动推送报告到微信 |

## 快速开始

### 环境要求
- Windows系统
- Python 3.13+
- 网络连接

### 安装步骤

**Step 1: 安装Python依赖**

打开命令行，运行：
```bash
pip install requests beautifulsoup4 lxml python-docx
```

**Step 2: 配置MCP工具（可选，推荐）**

推荐安装以下工具，可获得更精准的搜索结果：
```bash
uv tool install ddg-search-mcp
uv tool install weixin-search-mcp
```

> 不安装也能运行，系统会自动降级到内置搜索

**Step 3: 运行程序**

```bash
python 老梨×光伏产业监控实例.py
```

## Skill详解

本项目包含3个核心Skill：

### Skill 1: 价格数据收集

**工作流程（8步）**

```
Step 1: 确定品种
Step 2: 列出候选信源（至少3次搜索）
Step 3: 评估信源（权威性/完整性/可得性）
Step 4: 确定主信源 & 补齐信源
Step 5: 数据收集
Step 6: 交叉验证
Step 7: 水位分析
Step 8: 输出数据
```

**核心概念**

- **主信源**：权威性最高的信源（如行业协会、政府机构、交易所），必须满足3年+数据覆盖
- **补齐信源**：辅助信源，用于填补主信源缺失
- **交叉验证**：主+主=强验证、主+补=中验证、补+补=弱验证

### Skill 2: 水位分析

**分析流程**

1. 收集数据 - 读取历史价格数据（3年+）
2. 计算百分位 - P20/P40/P60/P80
3. 判断水位 - 对照当前价格所在百分位区间
4. 输出结论 - 历史低位/低水位/中水位/中高水位/高水位

**百分位法**

| 当前价格位置 | 水位判断 |
|-------------|---------|
| > P80 | 高水位 |
| P60 ~ P80 | 中高水位 |
| P40 ~ P60 | 中水位 |
| P20 ~ P40 | 低水位 |
| < P20 | 历史低位 |

**输出格式**
```
数据周期：2023-01 ~ 2026-03
当前价格：4.32 万元/吨
历史区间：3.44 ~ 26.00 万元/吨
百分位：P20=4.32 P40=5.39 P60=6.00 P80=8.50
水位判断：历史低位
```

**评级逻辑（3档）**

| 评级 | 条件 | 结论 |
|------|------|------|
| 5星王者 | 产能>200GW + 低水位/历史低位 | 供需改善预期强烈 |
| 4星战神 | 产能100-200GW + 低水位/历史低位 | 供需边际改善 |
| 3星暂时关注 | 其他情况 | 需持续观察 |

### Skill 3: LL-负面舆情监控

**工作流程（7步）**

```
Step 1: 搜索策略（企业+负面词）
Step 2: 数据源搜索（ddg + weixin）
Step 3: 负面词匹配
Step 4: 信源筛选
Step 5: 信源核实（交叉验证）
Step 6: 风险判定
Step 7: 报告输出
```

**负面词库（8类41词）**

- 事故类：发生火灾、发生爆炸、起火、坍塌、倒塌
- 停产类：停产、停工、停业、停止作业、停止建设
- 监管类：被查、被立案、被责令整改、被处罚、吊销、关停、取缔
- 电力类：限电、停电、断电
- 供应链类：断供、断货、供应中断
- 设备类：故障、损坏、报废、停机
- 资金类：资金链断裂、破产、债务违约、资不抵债
- 灾害类：洪水、台风、暴雨、地震

**信源等级**

| 等级 | 类型 | 代表信源 |
|:----:|------|---------|
| A | 官方公告 | 巨潮、上交所、深交所 |
| B | 主流财经媒体（原创采编） | 财联社、证券时报、第一财经、澎湃新闻、界面新闻 |
| C | 行业垂直媒体 | 索比、黑鹰、PV Tech、微信公众号 |
| D | 内容聚合/自媒体 | 163.com、sohu.com、sina.com、ifeng.com、eastmoney.com、雪球、今日头条 |

> 注意：B级必须是**原创采编媒体**，内容聚合平台（如163.com、sohu.com等）虽然也有财经内容，但因UGC占比高，质量和可信度较低，划入D级。

## 常见问题

### Q: 运行报错怎么办？
A: 
1. 确认Python已正确安装：`python --version`
2. 确认依赖已安装：`pip list`
3. 查看错误日志，按提示排查

### Q: 搜索不到数据怎么办？
A: 检查网络连接，或等待一段时间后重试

### Q: 微信推送失败怎么办？
A: 确认微信MCP已正确配置，或查看日志文件
